Welcome to the WebPG Map Tool! This tool generates the necessary data structures for WebPG to use.

Enter the numeric dimensions of the map into the correct input fields. Then, enter the characters you will use in the map and the corresponding labels for those characters.

Map Canvas

The input canvas will appear after you input the required information!

Now 'draw' your map using the characters you entered in the 'characters' field.

Tip: resize the input area so that one row contains the same number of characters as the 'mapWidth' is set to.
